# Simple List of Notification

This is react native app built up with redux Saga and redux toolkit.

# How to run

1. Install dependencies

```
yarn install
```

2. Run android

```
npx react-native run-android
```

3. Run iOS

```
cd ios && pod install && cd ..
npx react-native run-ios
```
